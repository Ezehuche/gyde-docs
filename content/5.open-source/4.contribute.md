## General

### Does Gyde work with all Shopify themes?

The **Floating Popup** (App Embed) works with all Shopify themes. The **Inline Section** and **Product Page Teaser** blocks require an Online Store 2.0 theme. See [Theme Compatibility](/open-source/self-hosting) for details.

### How many quizzes can I create?

There is no hard limit on the number of quizzes. You can create, publish, and embed as many quizzes as your plan allows.

### Can I have multiple quizzes on the same store?

Yes. You can publish multiple quizzes and embed each one in different places — a skincare quiz on one page, a supplement quiz on another. The floating popup can be configured to show a specific quiz via its handle, or to automatically cycle through all published popup quizzes.

### Does the quiz work on mobile?

Yes. The quiz builder and all embed types (popup, inline, modal, full-page) are fully responsive and optimised for mobile shoppers.

## Product Recommendations

### How does Gyde decide which products to recommend?

Gyde looks at every choice the shopper selected. For each choice that has mapped products, one product is randomly selected from that choice's list. The resulting set of products (deduplicated) appears on the results page. See [Product Mapping](/features/submissions) for full details.

### Can I show the same product for multiple choices?

Yes. You can map the same product to as many choices as you like. If it's selected via multiple choices, it will appear only once on the results page.

### Can I show more than one product per choice?

Yes — map multiple products to a choice and Gyde will pick one at random each time that choice is selected. Mapping more products increases variety across sessions.

### What happens if a shopper's choices have no mapped products?

The results page will show the "no recommendations" state. To avoid this, make sure every meaningful choice option has at least one product mapped to it.

## Analytics

### Does Gyde track which products were recommended?

Yes. The submission log records which products were shown on the results page for each session.

### Does Gyde track Shopify orders placed after taking a quiz?

Add-to-cart events from the results page are tracked directly by Gyde. For downstream order attribution, the quiz ID and session ID are saved as line item properties on every cart item added through the results page — you can filter and report on these in your Shopify admin.

## Embeds

### Can I use the quiz outside of Shopify?

Gyde is built specifically for Shopify and its product recommendation features (fetching catalogue data, adding to cart, session tracking) require a Shopify storefront. The quiz form itself can be shared via its public URL, but the recommendations and cart features will only work on a Shopify domain.

### Can I trigger the quiz programmatically?

Yes. Once the Gyde embed script is loaded (via the App Embed block), you can open the popup from your own JavaScript using:

```js
window.Gyde.open()       // opens the default popup quiz
window.Gyde.open('your-quiz-handle')  // opens a specific quiz by handle
```
