Gyde is designed with shopper privacy in mind. This page explains what data is collected, how it's used, and what controls you have as a merchant.

## Data collected by Gyde

### Quiz answers

When a shopper submits a quiz, their answers are stored and associated with the quiz session. This data is:

- Visible to you (the merchant) in the **Submissions** tab.
- Used to calculate product recommendations.
- Never sold to third parties.
- Retained until you delete the submission or uninstall the app.

### Session data

Each time a shopper opens a quiz, Gyde creates an anonymised session record that includes:

- The device type (mobile, tablet, desktop)
- The page where the quiz was triggered
- The trigger type (popup, section, teaser)
- Whether the session ended in a submission

**No personally identifiable information** is captured at the session level unless the shopper provides it as a quiz answer (e.g. an email field).

### Add-to-cart events

When a shopper adds a product to cart from the results page, Gyde records the event linked to the session. The quiz ID and session ID are also written as **line item properties** on the Shopify cart item — these appear on the order in your Shopify admin.

## Shopify permissions

Gyde requests the following Shopify API permissions:

| Scope | Purpose |
|---|---|
| `read_products` | Fetch your product catalogue for the product mapping interface and recommendations |
| `write_products` | (Reserved for future features) |
| `read_themes`, `write_themes` | Install and update the quiz widget asset on your theme |
| `read_customers`, `write_customers` | Store quiz session data linked to customer profiles where available |
| `read_orders` | (Reserved for future attribution features) |
| `read_content`, `write_content` | Create embed pages when using the guided page creation feature |

## Shopper rights

Shoppers whose quiz answers are stored can request deletion of their data. As the merchant, you are the data controller — you can delete individual submissions from the **Submissions** page at any time.

## Data retention

- **Quiz submissions** are retained until deleted by the merchant or the app is uninstalled.
- **Session data** is retained for 12 months by default.
- **All data is deleted** within 30 days of app uninstallation.
