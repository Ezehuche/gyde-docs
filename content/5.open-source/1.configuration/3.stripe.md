The **Full Page** embed creates a dedicated Shopify page for your quiz at `/pages/quiz-your-quiz-handle`. The quiz fills the entire page, giving shoppers an immersive, distraction-free experience. Link to the page from your navigation, product pages, email campaigns, or ads.

## How to generate the page

1. Open the quiz in the Gyde app.
2. Go to **Quiz → Share → Embed Guide**.
3. Select the **Full Page** tab.
4. Click **Generate Quiz Page**.

Gyde calls the Shopify API to create (or update) the page immediately. Once generated, the page URL is shown in the Embed Guide — copy it to link from anywhere on your store.

## URL format

Generated pages follow the pattern:

```
/pages/quiz-your-quiz-handle
```

For example, a quiz with handle `skin-type-quiz` creates the page at `/pages/quiz-skin-type-quiz`.

## Requirements

The **Gyde Widget** App Embed must be enabled in your Shopify Theme Editor (`Online Store → Themes → Customize → App embeds → Gyde Widget`). The embed script loads on every page — including the generated quiz page — via the App Embed.

## Regenerating the page

If you rename the quiz (changing its handle) or want to reset the page content, click **Regenerate page** in the Embed Guide. This overwrites the existing Shopify page.

## Linking to the page

Add the quiz page URL to:

- **Navigation** — go to **Online Store → Navigation** and add the URL to your main menu.
- **Product pages** — add a "Take the quiz" link in your product description or theme blocks.
- **Email / SMS campaigns** — link directly to `/pages/quiz-your-handle` in campaign copy.
- **Ads** — use the full storefront URL (e.g. `https://your-store.myshopify.com/pages/quiz-skin-type-quiz`) as the landing page URL.
