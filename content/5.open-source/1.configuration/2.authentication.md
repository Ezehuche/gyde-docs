The **Gyde Section** block lets you embed a quiz inline on any page of your Shopify 2.0 store — the quiz becomes part of the page content, not an overlay. This is ideal for dedicated quiz landing pages, homepage sections, and collection pages.

## Requirements

- Your Shopify theme must support **Online Store 2.0** app blocks. Most themes released after 2021 do. If the section block doesn't appear in the Theme Editor, your theme may be a legacy theme — see [Legacy themes](#legacy-themes).

## Add the Section block

1. In your Shopify admin, go to **Online Store → Themes → Customize**.
2. Navigate to the page where you want to add the quiz (e.g. the homepage, a custom page, or a collection page).
3. Click **Add section** in the left panel.
4. Search for **Gyde Section** and select it.
5. Configure the settings and click **Save**.

## Settings reference

### Quiz handle

The handle of the quiz to embed (e.g. `skin-type-quiz`). This field is **required** — the block will show a placeholder if left blank.

Find your quiz handle in the Gyde app at **Quiz → Settings → Widget → Quiz handle**.

### Display type

Controls how the quiz renders in the section:

| Option | Description |
|---|---|
| **Standard** | The quiz renders as a full inline iframe directly on the page |
| **Modal** | A trigger button appears on the page; clicking it opens the quiz in a centred modal dialog |
| **Full-page** | The quiz takes over the full browser viewport when opened |

For most inline section use-cases, **Standard** is the right choice.

### Section height (Standard mode only)

Set the pixel or viewport-height value for the iframe. The quiz will scroll within this container if its content is taller.

## Positioning the section

The Gyde Section can be placed anywhere a standard Shopify section block can go — between the hero and the featured collection, at the bottom of a product page, or as the primary content on a dedicated `/pages/quiz` page.

Drag the section card in the Theme Editor's left panel to reorder it relative to other sections.

## Legacy themes

If your theme does not support Online Store 2.0 and the section block is unavailable, you have two options:

1. **Use the Floating Popup** — the App Embed works on all Shopify themes and requires no block support.
2. **Embed via a Custom HTML section** — add a Shopify **Custom Liquid** or **Custom HTML** section and paste the quiz iframe URL:
   ```html
   <iframe
     src="https://your-gyde-domain.com/f/YOUR_QUIZ_ID"
     width="100%"
     height="600"
     frameborder="0"
     allow="microphone; camera"
   ></iframe>
   ```
   Replace `YOUR_QUIZ_ID` with the quiz's ID (visible in the quiz URL inside the Gyde app).
