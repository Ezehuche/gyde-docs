Quiz settings are divided into two areas: **General** (submission behaviour and access controls) and **Widget** (Shopify storefront appearance and embed options).

Open settings by clicking the **Settings** button (gear icon) in the quiz builder toolbar, or from the quiz card's three-dot menu on the dashboard.

---

## General settings

### Progress bar

Show a progress indicator at the top of each question slide so shoppers can see how far along they are.

### Navigation arrows

Enable back/forward arrows so shoppers can revisit previous questions.

### View all questions

Allow shoppers to see a scrollable list of all questions and jump to any slide directly.

### Close form

Temporarily deactivate the quiz. Shoppers who visit the quiz URL or trigger the widget will see a closed/inactive message.

### Expiration date

Set a start and end date for the quiz. The quiz will only be accessible during that window — useful for seasonal campaigns.

### Maximum submissions

Limit the total number of times the quiz can be submitted. Once reached, the quiz closes automatically.

## Widget settings

### Quiz handle

A unique, URL-friendly identifier for the quiz (e.g. `skin-type-quiz`). Auto-generated from the quiz title, but you can edit it.

The handle is used to:

- Pin this specific quiz to the **Floating Popup** App Embed block in the Shopify Theme Editor.
- Reference the quiz in **Inline Section** blocks on specific pages.

::alert{type="info" icon="lucide:info"}
  Handles must be unique across all quizzes in your store. If you change the handle, update any Theme Editor blocks that reference the old one.
::

### Display type

Controls how the quiz opens when triggered:

| Display type | Behaviour |
|---|---|
| **Popup** | Opens in a floating overlay panel. Shows the mid-quiz product carousel between questions. |
| **Standard** | Renders inline on the page, embedded directly in the content area. |
| **Modal** | Opens in a centred modal dialog with a dark backdrop. |
| **Full-page** | Takes over the full browser viewport. |

### Bubble colour

The background colour of the floating bubble button used in popup mode.

### Bubble shape

- **Circle** — a round icon button (default).
- **Rectangular banner** — a wider pill-shaped button with text, suitable for text-heavy CTAs.

### Widget position

Where the floating bubble appears: **bottom-right** (default) or **bottom-left**.

### Teaser message

Short text shown in the teaser tooltip that pops up above the bubble to catch shopper attention (e.g. "Not sure which product is right for you?").

### Show teaser on page load

When enabled, the teaser tooltip appears automatically after a short delay on every page. When disabled, it only appears when a shopper adds something to their cart — a softer, more contextual nudge.

## Protection

### Password protection

Require shoppers to enter a password before they can view or submit the quiz.

### Bot prevention

Enable CAPTCHA to prevent automated submissions.
