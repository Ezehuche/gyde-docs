Every time a shopper completes a quiz, their answers are saved as a **submission**. You can view, search, filter, and export all submissions from the **Submissions** tab inside each quiz.

## Viewing submissions

Go to **Quiz → Submissions** to see a chronological list of all responses. Each row shows:

- Submission date and time
- A summary of the answers given
- Whether a product was added to cart from the results page

Click any submission to open a **Submission Detail** panel with the full set of answers, the session metadata (device, trigger page), and the products that were recommended.

## Searching and filtering

Use the search bar to find submissions containing specific answer text. You can also filter by:

- **Date range** — narrow down to a specific period.
- **Has add-to-cart** — show only sessions where a product was added to cart.

## Exporting submissions

Click **Export** to download submissions as a CSV file. The export includes all answer columns, session metadata, and any recommended products.

## Deleting submissions

Select one or more submissions using the checkboxes, then click **Delete** in the toolbar. Deleted submissions are permanently removed.

::alert{type="warning" icon="lucide:triangle-alert"}
  Deleted submissions cannot be recovered. Export a backup before deleting in bulk.
::

## Spam filtering

Gyde automatically flags suspected bot submissions based on timing patterns and other signals. Click **Inbox → Spam** to review flagged submissions. You can mark a submission as "Not spam" to move it back to the main inbox.
