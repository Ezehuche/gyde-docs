Gyde gives you four ways to embed a quiz on your Shopify store. Open **Quiz → Share → Embed Guide** in the Gyde app for a step-by-step wizard that generates the exact setup instructions for your theme.

| Method | Best for |
|---|---|
| **Floating Popup** | A bubble that appears on every page — site-wide quiz access |
| **Inline Section** | Embedding the quiz as part of a page's content (homepage hero, landing page, collection page) |
| **Modal Button** | Opening the quiz via any existing button or link in your theme — no Theme Editor block needed |
| **Full Page** | A dedicated Shopify page at `/pages/quiz-your-quiz-handle` that you can link from navigation or campaigns |

---

## Floating Popup

The **Gyde Widget** App Embed adds a floating bubble to every page of your store. Shoppers click the bubble to open the quiz in an overlay without leaving the page.

### Setup

1. In the Gyde app, go to **Quiz → Share → Embed Guide** and select the **Floating Popup** tab.
2. Choose your theme from the dropdown and click **Open App Embeds**.
3. In the Shopify Theme Editor, toggle **Gyde Widget** on.
4. Paste your quiz handle into the **Quiz handle** field, configure the bubble style, then click **Save**.

### Settings in Theme Editor

| Setting | Description |
|---|---|
| **Quiz handle** | Handle of the quiz to open (e.g. `skin-type-quiz`). Leave blank to load all published popup quizzes. |
| **Show widget on** | **All pages** or **Product pages only**. |
| **Hide bubble on product pages** | When on, the bubble stays hidden until a shopper adds a product to cart. |
| **Bubble shape** | **Circle** — round icon button. **Rectangular banner** — pill-shaped button with text. |
| **Widget position** | **Bottom right** or **Bottom left**. |
| **Bubble color** | Background fill of the bubble (default: `#6366f1`). |
| **Show teaser on page load** | When on, a tooltip appears above the bubble after a short delay. When off, the teaser only appears when a shopper adds to cart. |
| **Teaser message** | Text in the teaser tooltip (e.g. "Not sure which product is right for you?"). |
| **Button text** | Label on the rectangular banner shape (e.g. "Find Your Perfect Match"). |

## Inline Section

The **Gyde Section** block embeds the quiz inline on any page — the quiz becomes part of the page content, not an overlay.

### Setup

1. In the Gyde app, go to **Quiz → Share → Embed Guide** and select the **Inline Section** tab.
2. Choose your theme and click **Open Theme Editor**.
3. In the Theme Editor, navigate to the page where you want the quiz (homepage, collection, custom page).
4. Click **Add section → Gyde Section**.
5. Paste your quiz handle into **Quiz Handle**, adjust sizing, then click **Save**.

Requires a Shopify Online Store 2.0 theme. See [Theme Compatibility](/open-source/self-hosting) if you're on a legacy theme.

### Settings in Theme Editor

| Setting | Description |
|---|---|
| **Quiz Handle** | Handle of the quiz to embed (required). |
| **Full width** | Stretches the embed to fill the entire browser width, breaking out of the theme's content container. |
| **Minimum width** | Minimum width in pixels (0 = fills container). |
| **Full height** | Stretches the embed to fill the full viewport height. |
| **Minimum height** | Minimum height in pixels when Full height is off (default: 600 px). |
| **Corner radius** | Rounds the iframe corners (0–24 px). |

## Modal Button

Open the quiz in a modal overlay when a shopper clicks **any** button or link in your theme — no Theme Editor block required.

### Setup

1. In the Gyde app, go to **Quiz → Share → Embed Guide** and select the **Modal Button** tab.
2. Copy the **element ID** shown (format: `gyde_your-quiz-handle`).
3. In your Shopify theme, add that ID to any button or link element in your theme code or a Custom HTML section:

```html
<!-- Button example -->
<button id="gyde_skin-type-quiz" type="button">Take the quiz</button>

<!-- Link example -->
<a href="#" id="gyde_skin-type-quiz">Take the quiz</a>
```

The Gyde embed script (loaded via the **Gyde Widget** App Embed) automatically detects the ID and wires up the click handler — no extra JavaScript needed.

::alert{type="info" icon="lucide:info"}
  The **Gyde Widget** App Embed must be enabled in your Theme Editor for the embed script to load and the click handler to work.
::

## Full Page

Gyde creates a dedicated Shopify page at `/pages/quiz-your-quiz-handle`. The quiz fills the entire page, giving shoppers an immersive, distraction-free experience.

### Setup

1. In the Gyde app, go to **Quiz → Share → Embed Guide** and select the **Full Page** tab.
2. Click **Generate Quiz Page**. Gyde creates (or updates) a Shopify page for this quiz.
3. The page is live immediately at `/pages/quiz-your-quiz-handle`.
4. Link to the page from your navigation, product pages, email campaigns, or ads.

::alert{type="info" icon="lucide:info"}
  The **Gyde Widget** App Embed must be enabled in your Theme Editor for the embed script to load on the generated page.
::

## Finding your quiz handle

Every quiz has a handle auto-generated from its title (e.g. "Skin Type Quiz" → `skin-type-quiz`). Find and copy it from **Quiz → Settings → Widget → Quiz handle** in the Gyde app.
