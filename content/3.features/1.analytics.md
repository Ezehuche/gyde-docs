The quiz builder is where you create and edit your product recommendation quizzes. It uses a slide-based layout — each question fills one screen — keeping the shopper experience focused and conversational.

## Question types

| Type | Description |
|---|---|
| **Multiple choice** | Shoppers pick one or more options from a text list. The most common type for product recommendations. |
| **Picture choice** | Same as multiple choice but each option has an image — great for visual products like apparel or decor. |
| **Short text** | A single-line text input for open-ended questions (e.g. name, email, specific concerns). |
| **Long text** | A multi-line text area for detailed open-ended answers. |
| **Statement** | A slide with no input — use it for welcome screens, section headers, or explanatory copy between questions. |
| **Thank you** | The final slide shown after submission. You can customise its heading and description. The results page replaces this slide for Shopify shoppers. |

## Adding and editing questions

Click the **+** button in the left panel to insert a new question. Select a question slide to edit its title and, for choice questions, its option labels.

You can drag question cards up or down in the left panel to reorder them at any time.

### Options Panel

While a question is selected, the **Options Panel** on the right side lets you:

- **Required / Optional** — toggle whether the shopper must answer before advancing.
- **Insert image** — attach a photo or illustration to the question slide.
- **Character / choice limits** — set minimum and maximum selection counts for choice questions.
- **Conditional logic** — set rules that show, hide, or jump to other questions based on previous answers. See [Conditional Logic](/features/form-settings).

## Form design

Click **Design** in the top toolbar to customise the visual appearance of your quiz:

- **Theme** — choose a colour palette and font that match your brand.
- **Button colour** — controls the primary action button, progress bar, and selected choice highlights.
- **Custom CSS** — inject your own CSS rules for full control over the quiz appearance. See [Custom CSS](/features/embed-your-form).

## Publishing

All changes save automatically. When your quiz is ready, click **Publish** in the top-right corner.

- **Published** quizzes are live and can be embedded or accessed by shoppers.
- **Unpublished** quizzes are invisible to the storefront — use them to draft new quizzes or safely edit live ones.

You can unpublish a quiz at any time from the dashboard's three-dot menu.

## Duplicating and deleting quizzes

From the dashboard:

- **Duplicate** — creates an identical copy of the quiz (useful for A/B testing different quiz flows).
- **Delete** — moves the quiz to the Trash. Quizzes in the Trash can be restored within 30 days or permanently deleted.

::alert{type="warning" icon="lucide:triangle-alert"}
  Deleting a quiz removes its product mappings and analytics history. This cannot be undone after the 30-day recovery window.
::
