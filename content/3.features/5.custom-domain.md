Gyde automatically tracks shopper activity across all your quizzes. No third-party tools or extra setup required.

## Quiz-level overview

Open any quiz in the Gyde dashboard and click the **Analytics** tab to see:

| Metric | Description |
|---|---|
| **Views** | Number of times the quiz was opened by a shopper |
| **Completions** | Number of times the quiz was fully submitted |
| **Completion rate** | Completions ÷ Views, expressed as a percentage |
| **Average duration** | Average time shoppers spend from the first question to submission |
| **Add to cart** | Number of products added to cart from the results page, attributed to this quiz |

## Session tracking

Every time a shopper opens a quiz, Gyde starts a **session**. Sessions capture:

- The **trigger type** (popup, section embed, product page teaser)
- The **page** where the quiz was opened
- The **device type** (mobile, tablet, desktop)
- Whether the session ended in a submission

Sessions are anonymised — no personally identifiable information is stored unless the shopper provides it as a quiz answer.

## Add-to-cart attribution

When a shopper adds a product to cart from the results page, Gyde records the event and links it back to the session and quiz. This lets you see which quizzes are driving direct cart additions.

Each cart item is also tagged with `_gyde_quiz_id` and `_gyde_session_id` as Shopify line item properties. This means you can filter quiz-attributed orders in your Shopify **Orders** page or export them via the Shopify admin API.

## Per-question report

The **Report** tab shows how shoppers answered each question:

- For choice questions: a bar chart of how often each option was selected.
- For text questions: a list of responses.

Use this data to identify drop-off points, popular choices, and which answer paths lead to add-to-cart events.

## Submission log

Go to **Quiz → Submissions** to view a full log of every quiz submission. Each entry shows:

- The date and time of submission
- All answers given
- The products recommended on the results page
- Whether the shopper added any products to cart

You can filter by date range, search by answer content, or export submissions as CSV.
