Gyde's built-in theme editor covers colours, fonts, and button styles. For deeper visual customisation — custom layouts, animations, typography overrides, or brand-specific components — you can inject your own CSS directly into the quiz.

## How to add custom CSS

1. Open the quiz in the quiz builder.
2. Click **Design** in the top toolbar.
3. Scroll to the **Custom CSS** field at the bottom of the design panel.
4. Enter your CSS rules and they will apply immediately in the preview.

## Useful selectors

The quiz renders inside a shadow-free iframe, so standard CSS selectors work without any scoping issues. Some commonly targeted elements:

```css
/* Question title */
.heyform-title { font-family: 'Your Font', sans-serif; }

/* Choice option button */
.heyform-choice-item { border-radius: 12px; }

/* Selected choice state */
.heyform-choice-item.selected { border-color: #your-brand-color; }

/* Primary action button (Next / Submit) */
.heyform-submit-button { border-radius: 999px; text-transform: uppercase; }

/* Progress bar */
.heyform-progress-bar { height: 6px; }
```

## Tips

- Use your browser's DevTools to inspect the quiz iframe and discover the exact class names in use.
- Custom CSS applies to the quiz everywhere it is embedded (popup, inline section, modal).
- Avoid overriding layout-critical properties like `position`, `z-index`, or `overflow` on the root container, as this can break the quiz flow.

::alert{type="warning" icon="lucide:triangle-alert"}
  Custom CSS requires familiarity with CSS. Changes that break the quiz layout are your responsibility to debug. If you get stuck, remove the custom CSS and the quiz will revert to its default appearance.
::
